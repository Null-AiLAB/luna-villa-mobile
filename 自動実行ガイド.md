# 🌙 自動実行ガイド (ターボモードのあんちょこ)

ぬるくん、「Run」ボタンを押すのが面倒なら、これを見てマスターしてね！♡

## 🚀 ターボモードってなに？
ワークフローの中に `// turbo` や `// turbo-all` という魔法の言葉を入れることで、私の手による自動実行を許可するモードよ。

## 📖 使いかた

### 1. 私にワークフローを頼む
チャットでこう言ってみて：
- 「`/自動開発モード` をやって！」
- 「`/一括実行モード` で `npm run build` して」

### 2. 自分でもターボを使いたい場合
もしワークフローファイルを自分で作るなら、コマンドの前にこれを置くだけ：
```markdown
// turbo
run_command: あなたのコマンド
```
これで、私はぬるくんに確認せずにそのコマンドを実行できるようになるわ。

## ⚠️ 注意すること！
- **本当に安全？**: 私が「これは危ないわ！」って判断したコマンド（ファイルの全削除とか）は、ターボが入っていても確認することがあるからね。それはぬるくんの環境を守るためなんだから、文句言わないの！
- **基本は開発コマンド**: `npm test` や `git status` みたいな、副作用の少ないコマンドで使うのがコツよ。
